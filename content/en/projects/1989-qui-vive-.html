GRRR 2015 CD audio, 1989

body: >-


"Provocative and fascinating." (Le Monde la Musique) A satire about television ( Zappeurs-Pompiers 2 ) and religion ( La calotte ). The cover by Italian cartoonist Massimo Mattioli (Squeak the mouse) is as crazy as the album. Jean-Jacques BIRGÉ (synthesizer, vocals, zapping, flute, etc) Bernard VITET (trumpets, vocals, reed, piano, etc) Francis GORGÉ (synthesizer, guitar, percussion, etc) A video live version of Zappeurs-Pompiers 2 is on YouTube (1h16).

This album is on BANDCAMP


n°1 / Neige / 3'49
Jean-Jacques Birgé - vocals, H3000 Bernard Vitet - flugelhorn Francis Gorgé - guitar

n°2 / L'improbable scénario / 5'35
Jean-Jacques Birgé - synthesizer, sampler Bernard Vitet - trumpet Francis Gorgé - sampler

n°3 / Sacrée soirée / 8'14
Jean-Jacques Birgé - sampler, H3000 Bernard Vitet - trumpet Francis Gorgé - sampler

n°4 / Raid sur le P.A.F. / 5'14
Jean-Jacques Birgé - field recording, sampler Francis Gorgé - sequencer

n°5 / Opéra savon / 4'04
Jean-Jacques Birgé - sampler, inanga Bernard Vitet - reed trumpet Francis Gorgé - guitar, sampler

n°6 / L'ai je bien descendue † / 8'36
Jean-Jacques Birgé - sampler, field recording, vocals Bernard Vitet - vocals Francis Gorgé - sampler, vocals

n°7 / Des haricots la fin / 8'34
Jean-Jacques Birgé - text, radiophonie (plunderphonics), field recording, flute Bernard Vitet - vocals, flute Francis Gorgé - programmation, flute Reissued in CD of the 10th Anniversary of Festival des 38e Rugissants.

n°8 / The Last Word / 3'12
Jean-Jacques Birgé - vocals, text Bernard Vitet - piano Francis Gorgé - rhythm

n°9 / Le chemin hanté / 2'33
Jean-Jacques Birgé - the animals Bernard Vitet - the voice of the wind, tenor violin Francis Gorgé - acoustic guitar, percussion

n°10 / Délits d'initiés / 6'26
Jean-Jacques Birgé - vocals, rhythm, flute Bernard Vitet - trumpet Francis Gorgé - electric guitar

n°11 / Cul par-dessus tête / 5'14
Jean-Jacques Birgé - flute Bernard Vitet - ritual trumpets, bendirs Francis Gorgé - congas, cymbals and the participation of fanatics

n°12 / Rupture d'anévrisme / 4'16
Jean-Jacques Birgé - string quartet, synthesizer Bernard Vitet - reed trumpet, vocals Francis Gorgé - electric guitar, vocals

n°13 / L'odeur du cirque / 2'10
Jean-Jacques Birgé - clarinet, harpsichord, merry-go-round Bernard Vitet - trumpet Francis Gorgé - saxophones, harp
