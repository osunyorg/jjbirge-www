"Si vous voulez expliquer le monde actuel à vos enfants, offrez-leur ce disque." (Jazz magazine, Disque d'émoi) "Manifeste pour une musique qui aurait du sens dans un monde qui l'a perdu." (Télérama) Neuf chansons qui font mal. Jean-Jacques BIRGÉ (voix, synthétiseurs, guimbarde) Bernard VITET (trompettes, anche, voix) Francis GORGÉ (guitares, ordinateur) Gérard SIRACUSA (percussion, voix)

Cet album est sur BANDCAMP


n°1 / Géno-Sida / 7'40
Jean-Jacques Birgé - voix, H3000, sampler Bernard Vitet - trompette, anche Francis Gorgé - guitare, séquenceur Gérard Siracusa - percussion

n°2 / Mémé / 5'45
Jean-Jacques Birgé - voix, synthétiseur, échantillonneur, mobylette-jouet Bernard Vitet - trompette Francis Gorgé - guitare Gérard Siracusa - percussion

n°3 / Cannibale / 7'29
Jean-Jacques Birgé - effets vocaux, H3000, synthétiseur Bernard Vitet - voix Francis Gorgé - guitare, séquenceur Gérard Siracusa - percussion

n°4 / Der Hass ist der Armen Lohn / 4'28
Jean-Jacques Birgé - voix, échantillonneur Bernard Vitet - trompette Francis Gorgé - guitare Gérard Siracusa - percussion

n°5 / Le mot le plus beau / 6'17
Jean-Jacques Birgé - voix, synthétiseur Bernard Vitet - anche, trompette Francis Gorgé - guitare Gérard Siracusa - percussion

n°6 / Ouir only inuit forza mon nez / 7'05
Jean-Jacques Birgé - guimbarde, sampler Bernard Vitet - anche, voix Francis Gorgé - guitare Gérard Siracusa - voix, percussion, séquenceur

n°7 / Les quatre vérités / 6'30
Jean-Jacques Birgé - voix, sampler, synthétiseur Bernard Vitet - trompette Francis Gorgé - guitare Gérard Siracusa - percussion

n°8 / Singulier possessif / 4'10
Jean-Jacques Birgé - sampler Bernard Vitet - voix Francis Gorgé - guitare basse Gérard Siracusa - percussion

n°9 / A French Letter / 6'31
Jean-Jacques Birgé - voix, synthétiseur, sampler Bernard Vitet - trompette Francis Gorgé - guitare Gérard Siracusa - percussion
